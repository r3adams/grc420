Renting a Domain Name

  1. Go to domain.com, Google Domains, Dreamhost, Namecheap, or another URL site.
  2. Search for the name you want to use, e.g., john-public. Note the various suffixes available and associated annual fees. In this example, “john-public.site” is only $0.99 for one year, without the extra-cost Privacy + Protection.
  3. Choose the domain name and suffix you want to use and check out.
  4. Go to Pointers and Subdomains and set the URL to your Github repository folder and HTML file you want viewers to see.
