Printing Standards

GRACoL—General Requirements for Applications in Commercial Offset Lithography, used for commercial printing, meaning sheefed offset. Also includes SWOP

SWOP—Specifications for Web Offset Publications, used for web offset printing on coated paper with heated drying, as for magazines and catalogs.

SNAP—Specifications for Nonheatset Advertising Printing, used for newspapers printed on web presses without heated drying.

FIRST—Flexographic Imaging Reproduction Specifications and Tolerances, used for flexo printing.

The 4 Cs model refers to a process for setting up and adjusting color reproduction equipment for the most accurate and consistent color, ensuring that each device performs at its best. The 4 Cs are depicted in a hierarchy where lower steps must be done first.

    Consistency

  • Choose an applicable print standard (table at left) and run to the specified ink densities.

    Calibration

  1. Calibrate the platesetter using the raster image processor (RIP) software and calibration routine.
  2. The calibration routine will image a series of tone scales. Measure the tones using a plate-reading densitometer set to read dot area.

    Characterization

  1. Print a printer profiling target and measure with an automated spectrophotometer.
  2. Make the ICC color profile from the measured data.

    Conversion

  1. Set the press profile as the output profile in the RIP.
  2. Or in Photoshop select Edit > Convert to Profile with the press profile.
